Вот краткие ответы на контрольные вопросы (ПМ.02):

1.ПК в ПМ.02 — проектирование БД, разработка объектов MySQL, создание интерфейса (tkinter), администрирование СУБД.

2.Почему Python + MySQL + tkinter?** — простота, кроссплатформенность, tkinter встроен в Python. *Альтернативы:* PyQt5, SQLite, C#+MSSQL.

3.ER-диаграмма — схема сущностей и связей. Нужна для визуализации структуры БД до реализации.

4.Типы связей: 1:1 (паспорт-сотрудник), 1:N (категория-товар), N:M (студенты-курсы).

5.Git — система контроля версий. `commit` — сохранение локально, `push` — отправка на удалённый сервер (GitHub).

6.Проверка Python и tkinter:** команды `python --version` и `python -m tkinter` (откроется окно).

7.Меры безопасности: не работать мокрыми руками, перерывы каждые 45 минут, блокировать экран при уходе (Win+L).

8.127.0.0.1 — localhost, адрес вашего же компьютера (для тестирования без сети).

9.Добавить файл в Git:`git add file.py` → `git commit -m "комментарий"` → `git push`.

10.README.md — описание проекта (цель, установка, запуск, авторы).
