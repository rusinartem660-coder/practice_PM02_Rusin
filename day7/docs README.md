# Управление интернет-магазином

## Описание проекта
Веб-приложение для управления интернет-магазином с авторизацией,
разграничением ролей (администратор/работник) и защитой от ботов (капча).

## Функциональность
- ✅ Авторизация с хешированием паролей (bcrypt)
- ✅ Защита от ботов (текстовая капча)
- ✅ Разграничение доступа по ролям
- ✅ Панель администратора (управление пользователями)
- ✅ Панель работника (ограниченный функционал)

## Технологии
- Python 3.8+
- Flask 2.3.0
- MySQL
- bcrypt
- HTML/CSS
